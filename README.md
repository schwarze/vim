vim
===
Portable vim configuration w/ self-installing plugins

Needed:
- git
- ctags (optional)

Install
==

OS X:
- cd ~
- git clone https://github.com/schwarze/vim.git ./vim
- ln -s vim/.vimrc .vimrc

Windows:
- cd %HOME%
- git clone https://github.com/schwarze/vim.git .\vim
- linkd vim\.vimrc .vimrc
